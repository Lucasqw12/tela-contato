# tela-contato
📩 Contact Form UI  Este projeto apresenta uma tela de contato moderna, Formulário para envio de mensagens Campos para nome e email Botão "Send Now" para envio Design limpo e responsivo Ícones ilustrativos para melhor experiência visual.
Este projeto apresenta uma tela de contato moderna, desenvolvida para permitir que usuários enviem mensagens facilmente através de um formulário simples e organizado.

🚀 Funcionalidades
Layout dividido em duas colunas
Seção com informações de contato (endereço, telefone e email)
Formulário para envio de mensagens
Campos para nome e email
Botão "Send Now" para envio
Design limpo e responsivo
Ícones ilustrativos para melhor experiência visual
🛠️ Tecnologias Utilizadas
HTML5
CSS3
Flexbox / Grid (layout)
Ícones (Font Awesome ou simil
